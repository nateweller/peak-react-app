# Peak React App

The front-end React application of the Peak ecosystem.

Built with `create-react-app` and `electron`.

## Quick Start

1. Clone the repository.
2. `yarn install`
3. `yarn start`

## Local Development

* `yarn start`

## Deploying for Web

* `yarn build`

## Deploying for Electron

1. `yarn build`
2. `yarn electron-package`
